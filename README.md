## Travel and Tourism
## Description
 A modern Travel and Tourism website designed to help users explore destinations, plan trips, and discover travel experiences with an elegant interface.
 This project showcases a responsive front-end travel platform with destination browsing, budget filtering, search functionality, and interactive UI elements. 

## Features
 - Destination Explorer – Browse travel destinations with images and details
 - Budget Filter – Find destinations based on travel budget
 - Search & Filter System – Search destinations by name, region, or category
 - Hero Image Slider – Attractive homepage slider showcasing travel spots
 - Live Chat UI – Simulated customer support chat interface
 - Statistics Section – Shows traveler statistics and ratings
 - Travel Experience Section – Highlights travel benefits and services
 - Gallery Strip – Travel photo gallery with scrolling animation
 - Login & Registration Modals – User authentication interface
 - Responsive Design – Works on desktop, tablet, and mobile

## Technologies Used

 -CSS3
 -JavaScript
 -Google Fonts
 -Unsplash Images
